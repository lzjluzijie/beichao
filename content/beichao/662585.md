&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;穿越集团的终极目标，从来就是一个口水不断，最终被务实的当权派小心回避的问题。&nbsp; &nbsp;从论坛开始时，你要夏威夷，我做塔西提，他要西班牙的嘴炮，到小说实际操作中，建立一个二十世纪富强民主文明的 XXX 主义和谐社会的行动方向，甚至包括论坛撕逼时类似，宅男屌丝元老在革命胜利后能吃穿不愁当个小富翁，就是集团莫大恩泽的言论。&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;本质上都涉及终极目标这个命题。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 我认为终极目标的混乱，来源于小说里穿越众的穿越动机，以及性格特质。（有什么样的性格特质，就决定了参加穿越的动机）

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 在我看来，至少可以分为以下几种：

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 1，征服者&nbsp; &nbsp;&nbsp; &nbsp;代表人物：&nbsp; &nbsp;文&nbsp;&nbsp;马&nbsp; &nbsp; （盗泉子）&nbsp; &nbsp;独孤雯&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;北美帮？&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;他们是为了征服一个新世界，建设自己心中理想的帝国而穿越的。通常有自己的一套世界观，有丰富的知识，强大的行动力，老辣的手腕。他们不太在意物质享受，对艰苦能够忍受，习惯于繁重的工作和责任，渴望权力，强烈希望能够在集团中成为领袖。最痛苦的就是，看着政敌建设敌人的理想帝国，而自己沦为看客。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 2，野心家&nbsp; &nbsp;&nbsp; &nbsp; 代表人物：&nbsp; &nbsp;马甲&nbsp; &nbsp;&nbsp; &nbsp;刘大市长

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 他们往往在现实中，有一定的能力地位，但现实事业的发展预期难以符合自己的野心。为了成为位高权重的人而参加大业。他们对终极命题，到底该建设一个什么样的未来，常常没有自己的一套理论。但他们需要在集团的发展中，实现自己成为人上人的野心，通常是政治野心。他们不在意艰苦，愿意接受有回报的繁重工作，他们认为物质享受体现了自己的政治成就所以并不排斥。他们渴望权力，并且尽一切努力展示自己，争取权力。&nbsp; &nbsp;他们可能不关心终极命题，因为他们自信自己的政治地位，肯定能确保他们的收获利益是一两人之下，几百人之上。他们鄙视目光短浅，鄙视安于现状，鄙视所谓捍卫元老权利的行动。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 3，技术狂人&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;代表人物：&nbsp; &nbsp;小白狼&nbsp; &nbsp;王若宾

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 具有技术专长，常常是某个至关重要的领域的权威，因而得到集团的重视和优待。为了实现自己的理想——建设巨舰，造大炮之类穿越。他们能够忍受艰苦，但希望有合理的舒适的生活质量，他们对于专业领域内的繁重工作甘之如饴，他们对物质需求相对要求不高。对于终极命题，他们往往没有兴趣，但潜意识里，他们明白因为自己对知识的垄断地位，决定了他们最终的利益分配必定是丰厚的，所以也无须过分关心。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 4，现实不如意，希望重新开始的普通人&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;代表人物：萧子山

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 有一些技术专长，但往往对穿越没有重要意义。个人的能力，知识，手腕往往一般，在原有世界处境不佳，前途渺茫，因而决定冒险一试。在穿越后，因为对自己能力的不自信，在初期往往并不出众，做些普通工作，随着集团发展，身边人飞黄腾达，开始意识到自己如不努力，恐怕会再次沦为群众。通常不太能忍受艰苦，不喜欢繁重的劳动，渴望舒适的生活。并不寄希望于天上掉馅饼，有朝一日直接提升当个总督。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 5，逼上梁山&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;代表人物：芳草地某个推了女学生的老师&nbsp; &nbsp;&nbsp;&nbsp;本人的同人

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 有一定技术专长，如果不出意外，往往未来前景不错。但由于某种灾难，为了逃避而临时穿越。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 6，被动穿越，嫁鸡随鸡&nbsp; &nbsp;&nbsp; &nbsp; 代表人物：&nbsp;&nbsp;某某表妹&nbsp; &nbsp;某某洋马&nbsp; &nbsp;某某女友&nbsp;&nbsp;某某老妈&nbsp; &nbsp;儿童

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 往往是没啥能力，也没啥野心的花瓶背板。 满足于现状，不关心未来。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 7，意外穿越&nbsp; &nbsp;&nbsp; &nbsp; 代表人物：&nbsp;&nbsp;海军提督一家&nbsp; &nbsp;郭大官人&nbsp; &nbsp; 兰度

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 但我还是认为有些人的穿越动机完全是不合理。比如河马医生，比如马疯子，吴南海，某台湾女人。
