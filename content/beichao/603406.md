，技术也不错，不知道为啥跑来穿越。

待人和气爱开玩笑，经常吹嘘执委到他面前，让执委张嘴执委就张嘴，让执委闭嘴执委就乖乖闭嘴。唯一一次骂人是有次喝醉了，他问候了

他原来医院领导的各位女性祖先。此外，他和大吧一样都是拜火教教徒————

&nbsp; &nbsp;&nbsp; &nbsp; 韩涛的顶头上司河马院长在对他的秘密鉴定里是这么写的：该医生作为著名医学学府的科班毕业生，理论基础扎实，专业技能熟练。

穿越后，由于医疗资源的紧张，在组织的要求下重新学习了全科医学的相关理论。目前，已能独立完成大内科常见疾病的诊治，手术技能尚待锻炼

。有多次海南各县巡回医疗队的带队经验。&nbsp; &nbsp;但似乎缺乏对医疗事业的奉献精神————

&nbsp; &nbsp;&nbsp; &nbsp; 在政保局的绝密元老鉴定中，是这么评价韩涛元老的：

&nbsp; &nbsp;&nbsp; &nbsp; 韩涛

&nbsp; &nbsp;&nbsp; &nbsp; 性别：男&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;年龄： 穿越元年&nbsp;&nbsp;31 岁

&nbsp; &nbsp;&nbsp; &nbsp; 技能：口腔医学专业----精通；&nbsp;&nbsp;医学专业-----中等

**第一节**

&nbsp; &nbsp;&nbsp; &nbsp; X 月 X 日&nbsp;&nbsp;星期五&nbsp; &nbsp;10：30am

&nbsp; &nbsp;&nbsp; &nbsp; 临高总医院口腔科

&nbsp; &nbsp;&nbsp; &nbsp; “ok，今天的治疗就到这。回去后，你的患牙疼痛症状一般在两三天内就会明显缓解了。不过，记得不要用这颗牙咬东西，饮食尽量

清淡，记得不要吃发性食物，比如牛羊肉之类。当然，在临高也没那么容易吃到。如果治疗后一两天内还有轻微疼痛，可以吃一片我给你开

的药。当然，如果仍然有剧烈的疼痛影响进食和睡眠的话，不要硬抗，请直接来找我。”

&nbsp; &nbsp;&nbsp; &nbsp;送走了这位元老患者，摘下不知道消毒多少遍的橡胶手套，拉开厚厚的棉布口罩，韩涛长出了一口气，在医师椅上伸了个懒腰，回头

问他的牙科助手兼学徒兼女秘书小房“后面还有病人吗？”

&nbsp; &nbsp;&nbsp;&nbsp;虽然好孩子读书会里的粗胚们时常用些“有事秘书干，没事干秘书”“要想会，先跟师傅睡”的粗俗笑话揶揄韩涛。可韩涛知道，自己

对这个女徒弟绝对是三分欣赏，两分敬佩，绝无一丝想法。毕竟，看着女徒弟健康的肤色，强健的体魄，憨厚的面容，韩涛暗暗觉得能对这

宝贝徒弟下手的人，绝对抵是三十五级的魔法师。没办法，当初选徒弟的时候，自己实在不想手把手的教一个抠脚大汉做口腔治疗，而医学

基础成绩尚可的女候选人只有两位，排除了拜师后不知道是自己照顾她还是她照顾自己的“林妹妹”，剩下的只有老实巴交的房二丫了。

&nbsp; &nbsp;&nbsp; &nbsp;“老师，今天上午没有预约的病人了，您可以休息一下。下午一点到三点，你要给医学生讲一堂病理大课，三点半到五点半，是口腔

学生的实训课，讲传导阻滞。晚上，读书会的元老们在农委会餐厅聚会。还有，河办回复，您的关于在济州岛建立寒带疾病研究所的建议，

河马院长觉得非常重要，他邀请了陆军代表和相关部门定在下周一下午两点院会议室开会，请您准时参加。”

&nbsp; &nbsp;&nbsp;&nbsp;“知道了，谢谢。”

&nbsp; &nbsp;&nbsp; &nbsp;午饭后，换了一身便装的韩涛，缓步行走在医院去往教室的路上。迎面过来各色人等，不住的和韩涛打招呼，有叫韩医生的，有叫韩

元老的，还有叫韩老师的。一边微笑着点头示意，韩涛的思绪却飘到了最近的时局上。

&nbsp; &nbsp;&nbsp; &nbsp; 临高暴恐事件发生后，众元老震惊于光荣伟大正确的 500 众穿越都六年了，早已打下了一份不小的江山，可在帝国的心脏，被誉为帝

国主人的元老却遭到了生命威胁。继而，随着案情的逐渐披露，有关部门明明已经掌握了重要线索，不但没有马上抓捕，为了所谓放长线钓

大鱼，竟然以不知情的元老为饵，坐视恐怖分子继续活动。最终，百密一疏，三个花样年华的小姑娘遇到了穷凶极恶手持利刃的恐怖分子，

仅仅因为钱朵朵的超水平发挥才没有酿成大祸。

&nbsp; &nbsp;&nbsp; &nbsp; 一时间，bbs 上民意沸腾，元老们纷纷主张立刻调查有关部门的处置是否有失误之处，主张加大对关乎元老切身利益的部门的监督。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 可两三天后，bbs 风向一变，几个从没漏过面的谁也不认识的 ID 跳出来，大骂在论坛上最活跃的元老院议长钱水廷，说他是阴谋颠覆现政权，破坏穿越众的事业，妄图用北美帮的独裁来代替现政权，是藏在革命群众内部的阴谋家大叛徒。旁人说句老钱说的有点道理，就被认定为大坏蛋钱逆水廷的党羽。

又过了几天，ｂｂｓ上又神秘的出现一群主张建立青年团的陌生人，大谈青年团的必要性之余，还大夸特夸钱朵朵，在这些人嘴里钱朵朵简直是仁义礼智信俱全，不世出的天才，言下之意，青年团总ｂｏｓｓ非钱朵朵莫属。韩涛旁敲侧击的问过身边元老，大家都不知道到底是谁在热捧青年团和钱朵朵。

ｂｂｓ上的波云诡谲让韩涛暗自心惊。　钱水廷贵为元老院议长，七执委之一，背后北美帮和宅党做后盾。可上面几下散手，就轻松批倒批臭，看来哪里的领导都不好惹，哪里的豺狼都吃人啊。韩涛不禁想起，ｂｂｓ上成功维稳后，那几个陌生的ｉｄ，得意之于，大谈领导和一线元老如何辛苦，如何了不起。还蹲在大后方的酱油元老都是废物，将来连个县长都不配当，革命胜利了，给笔钱让他们做个富家翁就是组织上的恩惠了云云。这到底是几个马仔的信口开河，还是——？

不管怎样，把自己的命运寄托在组织，寄托在领导的良心底线上，安心在大后方做本职工作，指望有朝一日，突然高官得作，骏马得骑肯定是不靠谱的。只有靠自己的奋斗，掌握了和上面讨价还价的筹码，才不会被轻松卖掉。心中下定决心，韩涛放开脚步大步向前。

**第二节**

傍晚

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 一身休闲的韩涛元老，信步步入被戏称为“好孩子读书会总舵“的农委餐厅乙二号包房。包房内先期到达的会员正三三两两的在一块聊天扯淡，桌子上摆着平元老从济州岛捎来的日韩点心小吃。服务员进来端给韩元老一杯绿茶并告诉韩涛今天的主菜是 海大官人赞助的烤乳猪套餐后，便退了出去。韩元老坐下来，挑了块看起来不错的点心放进嘴里，很快便融入某位长腿美女元老和两位青年军官的感情纠葛里。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;能得到平元老的”手信“，这倒不是说读书会是个多么重要的团体。自从萧主任和海军联合发起了向海外站点定期发送办公厅特供的冷船后，原本回程时空载或只能装些粗笨货物的冷船，便被有心元老用来向中央捎些地方土特产，东西没多贵重，就是表示个心意不是。时间一长，惯例一成，司长局长长老固然人人有份，为防被眼红的人说成是钻营，一些相熟的普通元老也能收到远方的礼物了。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp; 说起好孩子读书会，可以说是一个曾经辉煌又逐步没落的社团。想当年，穿越大业初兴，几百个血气方刚的小伙子困居百仞临高一隅，白天是枯燥沉重的体力劳动，晚上一回宿舍，便快到断电睡觉的时候，生活极端乏味。后来条件稍好，每人有了一点供电份额，可也还是不够用。一盏灯能照亮一人也可以照亮十人，于是有心人便开始收集大家零散的配额，集中起来包场，人们在那里可以联机，可以打牌，可以闲扯发发牢骚，当然也可以欣赏爱情动作片。读书会极盛时，曾有过五十人一同品鉴穗花的盛况。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;可随着时间的流逝，澳宋帝国的疆域日渐扩张，铁拳爆菊的威名响彻东亚，原来一起聊天打屁的元老，纷纷变身为科长局长司长站长营长参谋长。一些人自觉进入了统治阶级，不便再与草根一起流着口水盯着屏幕；一些人虽然尚未上位，但觉得身处体制内，要是一下班就跑去和一群宅男厮混，未免给领导此人缺乏上进心的不良印象，便逐渐退出。出席读书会聚会的人数越来愈少，聚会的间隔也越来越长，眼看读书会的家名便要断绝。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp; &nbsp;这时，突然出现了一则流言，据说读书会第三任召集人，暗中将宅男们在聚会聊天时无意中泄露的小道消息编辑成册，以私人信函的方式发给外省元老，外省元老也投桃报李以信息和礼物相谢。说的有鼻子有眼的，甚至传说圈内人将这些信件，戏称为”临高观察家报“。经过三四年政治洗礼的会员们，当然不会傻得去直接问召集人。不过，从那时起，读书会的餐桌上各种外地土特产确实是多了起来。

&nbsp; &nbsp;&nbsp; &nbsp;&nbsp;&nbsp;久而久之，有的会员渐渐淡出，但有的元老却又兴致勃勃的出现了。读书会的资金日渐宽裕，聚会也逐渐固定。甚至有几位元老，非常慷慨的不定期自掏腰包赞助聚会的套餐，这几位往往被戏称为”大官人“。
